游戏运营面试准备

Posted on 2017-08-11 | In game_operation

获得成就感主要途径：

  1. 打赢别人，比别人强
  2. 指挥大家，拥有权力和影响力
  3. 得到自己想要的东西
  4. 在团队中有归属感

怎么玩？->为什么玩？->下次还来吗?->如何玩的更好?->我要花钱吗?->我要玩到什么时候?

什么是运营？

拉新、留存、促活

  1. 深入理解产品，深入理解玩家
  2. 提供优质服务，解决玩家问题，服务好高付费玩家
  3. 维持稳定，包括服务器、数据的稳定
  4. 通过运营手段，包括活动、引导等，提高收入
  5. 改进产品，通过数据、用户反馈等发现问题，解决问题

活动

产品是活动的基础

活动可以尝试解决产品问题，测试效果

需求诞生的原因是不满足现状，需要通过一些途径实现自己的愿景。
